# Design System

## Theme
สว่าง ทันสมัย มีบุคลิกภาพที่น่าเชื่อถือ (Modern SDU Identity Light Mode) ผสานการใช้กระจกสะท้อน (Glassmorphism) เล็กน้อยเพื่อความลื่นไหลและดูระดับพรีเมียม

## Colors
- **Primary (หลัก)**: `#667eea` (คราม) - ใช้สำหรับเน้นส่วนสำคัญ ปุ่มหลัก และความก้าวหน้าหลัก
- **Primary Dark (หลักเข้ม)**: `#5568d3` - ใช้สำหรับสถานะโฮเวอร์และการควบคุมที่ต้องการเน้น
- **Secondary (รอง)**: `#a8edea` - ใช้เป็นเฉดเหลือบสีและเพิ่มมิติตกแต่ง
- **Success (สำเร็จ)**: `#16A34A` - ใช้เป็นสัญลักษณ์แทนความสมบูรณ์และถูกต้อง
- **Warning (เตือน)**: `#F59E0B` - แทนสถานะรออนุมัติหรือปรับแก้
- **Danger (ผิดพลาด/ยกเลิก)**: `#DC2626` - แทนสถานะปฎิเสธและยกเลิก
- **Light (สว่าง)**: `#F8FAFC` - พื้นหลังหลักของหน้าเว็บและแอปพลิเคชัน
- **Dark (เข้ม)**: `#1F2937` - ตัวหนังสือหลักและการควบคุมที่มีมิติคมชัด
- **Glass Panel (กล่องแก้ว)**: `rgba(255, 255, 255, 0.78)` พร้อม `backdrop-filter: blur(12px)`

## Typography
- **Font Family**: `'Sarabun', -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif`
- **Body Text**: ขนาด 1rem, น้ำหนักปกติ, ความยาวเหมาะสมสำหรับการอ่าน

## Layout & Border Radius
- **Border Radius**: `--radius: 12px`, `--radius-lg: 16px`
- **Box Shadow**:
  - `var(--shadow)`: `0 4px 6px rgba(0, 0, 0, 0.1)`
  - `var(--shadow-lg)`: `0 10px 25px rgba(0, 0, 0, 0.15)`
